# Notification Channels

## Telegram

ใช้ Telegram Bot API:
- Bot Token
- Chat ID

สถานะ: เสร็จ

## LINE

Sprint 1.8 เตรียม sender แบบ LINE Notify compatible/legacy หรือ proxy endpoint

หมายเหตุ: งานจริงควรพัฒนาเป็น LINE Messaging API เต็มรูปแบบ

สถานะ: โครงสร้างเสร็จ / Official Messaging API ยังไม่ทำ

## Email

เพิ่ม SMTP configuration แล้ว

ยังไม่ได้ดำเนินการ:
- ส่ง SMTP จริง
- เพิ่ม ESP_Mail_Client library
- TLS certificate handling

สถานะ: Config scaffold เสร็จ
